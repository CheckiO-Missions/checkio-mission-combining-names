将备受喜爱的明星情侣的名字组合成一个朗朗上口的速记符号，供大众媒体消费，原来是很容易实现自动化的。

首先，计算first 名字中存在多少个最大的连续元音组（aeiou，为使问题简单，字母y始终是辅音）。例如，"brad "和 "jean "有一个元音组，"jeanie"和 "britain "有两个元音组，"angelina"和 "alea x nder "有四个元音组。请注意，一个元音组可以包含一个以上的元音，如 "queueing "这个词就包含了整整一个五分音符。

如果名字只有一个元音组，则只保留该元音组前面的辅音，其他的都去掉。例如，"ben "变成 "b"，"brad "变成 "br"。否则，如果第一个单词有n> 1 个元音组，则保留倒数第二个元音组n- 1 之前的所有内容。例如，"angelina "变为 "angel"，"alexander "变为 "alex"。

将该字符串与去掉second 名称开头的所有辅音后得到的字符串进行连接。

输出： 字符串字符串 (str{% if is_js %}ing{% endif %}).
